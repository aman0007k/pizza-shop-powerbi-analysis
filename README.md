# pizza-shop-powerbi-analysis
This project focuses on analyzing pizza shop sales data to uncover meaningful business insights using Power BI. The dashboard provides a clear and interactive view of sales performance, customer demand, and product trends to support data-driven decision-making.
Pizza Shop Sales Analysis (Power BI)

This project analyzes pizza shop sales data using Power BI.

Key KPIs:
- Total Revenue
- Total Orders
- Best-selling Pizzas
- Peak Sales Time

Tools Used:
- Power BI
- DAX
- SQL
